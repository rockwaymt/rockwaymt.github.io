To add an image to a page:

  1. Find the image you want to use on this page. Make sure you Click to see it un-resized, click "back" to come back here

  2. Copy the location (the text below the image), do one of these:
    1. Easy copy:
      1. Right click the image
      2. Choose: Copy link location
    2. Manually copy:
      1. Select of the the text with the mouse cursor
      2. Right-click and choose Copy

  3. Go to the page you want to change
  4. Look for an <img src="blahBlahBlah">
  5. Delete the existing image location (everything between quotes)
  6. Right-click between the quotes, and choose Paste
  7. Save the page
